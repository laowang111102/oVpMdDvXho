# 学生就业管理系统 python1187

## 项目概述

学生就业管理系统是基于Python的Django框架开发的，旨在帮助学院、企业和学生三方进行高效的信息管理，以促进学生就业。系统包括学院信息管理、专业信息管理、企业信息管理、岗位信息管理以及学生信息管理等功能模块。

## 技术栈

- 后端：Python Django
- 前端：HTML, CSS, JavaScript
- 数据库：MySQL

## 功能模块

### 系统角色

#### 管理员功能

- 系统首页
- 学院信息管理
- 专业信息管理
- 企业信息管理
- 岗位信息管理
- 管理员信息管理
- 学生信息管理
- 学生教育经历管理
- 学生项目经历管理
- 学生投递记录管理

#### 招聘方功能

- 系统首页
- 岗位信息管理
- 学生信息管理
- 学生教育经历查看
- 学生项目经历查看
- 学生投递记录查看

#### 学生用户功能

- 系统首页
- 岗位信息浏览
- 教育经历设置
- 项目经历设置
- 投递记录查看

## 运行环境

- Python环境
- Django框架
- MySQL数据库

## 部署步骤

1. 安装Python环境
2. 安装Django框架
3. 配置MySQL数据库
4. 克隆或下载项目代码
5. 运行数据库迁移命令
6. 启动Django应用

## 核心亮点

- 支持学院、专业、企业、岗位等多维度的信息管理
- 面向不同角色的定制化功能，提高操作便捷性
- 帮助学生更好地管理自己的教育背景和项目经历，提高就业竞争力
- 系统设计简洁，易于维护与扩展
